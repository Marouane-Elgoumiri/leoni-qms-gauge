Vigilance Test - LEONI Operator Awareness Assessment

Vigilance Test Structure:

  • Test Information Section (Blue): Input form for creating new vigilance tests
  • • Name Defect: Text field for defect identification
  • • Responsable: Select box for the person responsible for the test
  • • Supervisor: Select box for the supervising person
  • • Line: Select box for production line (VOLVO MDEP, HDEP C1, etc.)
  • • Post: Select box for specific workstation/post
  • Test Results Table (Green): Displays all vigilance tests with results
  • • Result Column: Checkbox system for PASSED/FAILED status
  • Sample Data: Shows example entries with different test outcomes
